# 交付操作手册

## 1. 项目概况

本交付包对应当前最新的农业物联网管理平台版本，已包含以下能力：

- 设备接入、实时监控、历史分析
- 规则告警、通知、AI 诊断与报告
- 摄像头、抓图、图片时间轴
- 登录安全、只读 OpenClaw 接口
- 多租户底座、租户管理、租户初始化、租户运行配置
- 套餐/订阅底座与关键配额校验
- 媒体存储抽象与对象存储优先路径规范

## 2. 当前线上环境

### 2.1 访问地址

- 前端：`http://82.156.45.208`
- 健康检查：`http://82.156.45.208/healthz`
- Swagger：`http://82.156.45.208/docs/`

### 2.2 运行组件

线上当前为单机部署：

- `Nginx`
- `admin-api`
- `MySQL`
- `Redis`
- `admin-web` 静态资源

### 2.3 服务说明

- 对外访问端口：`80`
- 后端内部端口：`3001`
- MySQL：`3306`
- Redis：本机使用，不对外开放

## 4. 常用运维命令

### 4.1 服务状态

```bash
systemctl status agri-admin-api
systemctl status nginx
systemctl status mysqld
systemctl status redis
```

### 4.2 重启服务

```bash
systemctl restart agri-admin-api
systemctl restart nginx
systemctl restart mysqld
systemctl restart redis
```

### 4.3 查看日志

```bash
journalctl -u agri-admin-api -n 200 --no-pager
journalctl -u agri-admin-api -f
```

## 6. 设备接入说明

### 6.1 设备接口入口

设备端应访问：

```cpp
const char* API_HOST = "http://82.156.45.208";
```

### 6.2 设备 Token

设备接入 token 现在支持平台后台管理：

- 系统管理
- 设备接入凭证

租户级 token 在：

- 系统管理
- 租户管理
- 租户设备接入凭证

### 6.3 设备固件说明

当前交付包已包含 ESP32 稳定版固件源码：

- `soil_sensor_reporter.ino`

如果现场设备要切线上，请确认：

- `API_HOST`
- `API_TOKEN`
- WiFi 名称和密码
- 传感器寄存器参数

## 7. 多租户说明

当前系统已经具备以下多租户能力：

- 租户主档
- 租户登录入口
- 租户隔离查询
- 租户管理后台
- 租户初始化
- 租户运行配置
- 租户设备接入凭证
- 套餐/订阅底座

当前更适合的使用方式是：

- 先按“多租户技术底座”运行
- 暂不强推进订阅运营流程

## 8. 媒体与图片说明

当前媒体策略已调整为：

- 数据库存元数据
- 文件路径按对象存储风格组织
- 后续可平滑切腾讯云 COS / MinIO

当前推荐路径规范：

```text
tenant/{tenantCode}/camera/{cameraCode}/snapshot/{sourceType}/{YYYY}/{MM}/{DD}/{snapshotNo}.jpg
tenant/{tenantCode}/camera/{cameraCode}/snapshot/{sourceType}/{YYYY}/{MM}/{DD}/thumb/{snapshotNo}-thumb.jpg
```

当前还未强制接入腾讯云 COS，但底座已预留：

- `media_storage.storage_provider`
- `media_storage.public_base_url`
- `media_storage.bucket_name`
- `media_storage.region`
- `media_storage.path_prefix`

## 9. 系统设置建议

建议优先关注这些后台配置：

- 登录安全配置
- AI 调度
- 通知通道
- 网关配置
- 媒体存储

对于生产环境，建议：

- 登录限流保持启用
- 失败锁定保持启用
- 定期备份数据库
- 定期轮换设备 token
