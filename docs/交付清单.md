## 3. 当前交付重点功能

- P0 全部基础能力
- P1 基础版规则、自动控制策略、AI、系统设置
- P2 基础版设备影子状态、通知记录
- 指标字典页与指标接口
- 总览大屏多指标图表
- 基于指标字典的规则可视化编辑器
- 设备暂停/恢复检测链路
- 摄像头与媒体扩展 SQL 规划稿
