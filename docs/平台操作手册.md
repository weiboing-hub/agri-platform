# 平台操作手册

## 1. 项目说明

本系统为“智能农业环境监测与自动控制平台”后台，提供以下能力：

- 设备与区域管理
- 实时监控与历史分析
- 手动控制与控制记录
- 告警流转与通知记录
- 规则引擎与自动控制策略
- AI 诊断、AI 报告、AI 任务管理
- 指标字典与多指标总览图表
- 系统设置与审计日志

## 5. 常用操作

### 5.1 新增区域

1. 打开“区域管理”
2. 点击“新增区域”
3. 填写区域编号、名称、作物类型等信息
4. 点击保存

### 5.2 新增网关 / 传感器 / 执行器

1. 进入对应页面
2. 填写基础信息
3. 绑定所属区域与网关
4. 保存后回到列表查看

### 5.2.1 暂停或恢复设备检测

1. 打开“网关设备管理”
2. 在目标网关行点击“暂停检测”或“恢复检测”
3. 等待设备下次轮询命令
4. 查看“检测状态 / 期望状态 / 命令版本”是否更新

### 5.3 发起手动控制

1. 打开“手动控制”
2. 选择执行器
3. 选择控制类型和持续时间
4. 提交后到“控制记录”中查看执行结果

### 5.4 处理告警

1. 打开“告警中心”
2. 选择一条告警
3. 在详情区域执行：
   - 指派
   - 确认
   - 处理中
   - 关闭
4. 查看流转记录

### 5.5 发起 AI 诊断

1. 打开“AI诊断”
2. 填写触发方式、范围类型、范围 ID 和原因
3. 点击“发起诊断”
4. 查看诊断摘要、风险等级和建议动作

### 5.6 生成 AI 报告

1. 打开“AI日报 / 周报”
2. 选择报告类型和报告日期
3. 点击“立即生成”
4. 在详情区查看报告内容和版本信息

### 5.7 重试 AI 任务

1. 打开“AI任务队列”
2. 找到失败任务
3. 点击“重试”
4. 验证任务状态和关联报告变化

### 5.8 保存系统设置

1. 打开“系统设置”
2. 修改对应配置项
3. 若接入真实 AI，请填写“AI 模型服务”中的 `Base URL / API Key / 模型名称`
4. 若接入真实通知，请填写“通知通道配置”中的企业微信、钉钉或通用 Webhook 地址
5. 点击“保存设置”
6. 若需联调，可使用：
   - 测试 AI 服务
   - 测试通知
   - 测试设备连接

### 5.9 查看设备影子状态

1. 打开“设备影子状态”
2. 按区域、Shadow 状态、在线状态筛选
3. 查看 `desired / reported / drift / lastCommand`
4. 如需要，点击“重同步”

### 5.10 重发通知

1. 打开“通知记录”
2. 找到失败通知
3. 点击“重发”
4. 查看 `retry_count`、发送状态和响应摘要更新

### 5.11 维护指标字典

1. 打开“系统管理 -> 指标字典”
2. 新增或编辑指标编码、名称、单位、值类型
3. 配置正常范围、预警范围、图表颜色、排序
4. 保存后，规则引擎、自动控制策略、总览大屏会自动复用这些指标

### 5.12 查看多指标总览趋势

1. 打开“总览大屏”
2. 先看顶部统计卡
3. 在“多指标趋势看板”中点击指标卡
4. 主图会切换到当前指标趋势，底部柱状条表示补传占比

### 5.13 接入 ESP32 设备

1. 在 ESP32 固件中把 `API_URL` 指向：
   - `http://<server>:3001/api/v1/iot/ingest`
   - 或兼容旧地址 `http://<server>:3001/api/soil/ingest`
2. 保持 `Authorization: Bearer DgQHl5VE1HpU7ab0AxAvRgMuIOOXrA_I`
3. 固件上报当前温湿度 payload 后，平台会自动：
   - 建立默认接入区
   - 建立网关档案
   - 建立温度/湿度传感器
   - 写入历史读数
4. 上报成功后到以下页面查看：
   - 网关设备管理
   - 传感器管理
   - 实时监控
   - 历史分析

### 5.14 新增媒体节点

1. 打开“设备中心 -> 媒体节点管理”
2. 点击“新增节点”
3. 填写节点编号、主机地址、API 地址、RTMP/HLS 地址和 FTP 根路径
4. 保存后在列表查看健康状态和详情

### 5.15 新增摄像头

1. 打开“设备中心 -> 摄像头管理”
2. 填写摄像头编号、区域、网关、媒体节点、源流地址
3. 根据设备协议选择 `RTSP / RTMP / HTTP / FTP`
4. 保存后在详情区检查源流地址和转出地址

### 5.16 创建手动抓图

1. 打开“设备中心 -> 抓图任务”
2. 在“手动抓图”区域选择摄像头和抓图用途
3. 点击“立即抓图”
4. 到“历史图片时间轴”查看新增图片索引

### 5.17 接收 FTP 抓图元数据

1. 打开“设备中心 -> 抓图任务”
2. 在“模拟 FTP 接收”区域填写摄像头、FTP 路径和拍摄时间
3. 点击“写入 FTP 抓图”
4. 到“历史图片时间轴”按 `FTP 上传` 过滤查看结果
