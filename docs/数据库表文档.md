# 数据库表文档

## 1. 说明

本文档列出当前项目中的核心表、中文用途和关键字段，适合作为开发联调与交付说明。

## 2. 系统层

### `sys_users`

- 中文名称：用户表
- 用途：存储后台登录用户
- 关键字段：
  - `user_no` 用户编号
  - `username` 用户名
  - `real_name` 姓名
  - `password_hash` 密码哈希
  - `status` 状态

### `sys_roles`

- 中文名称：角色表
- 用途：定义角色模板
- 关键字段：
  - `role_code` 角色编码
  - `role_name` 角色名称
  - `role_level` 角色层级
  - `status` 状态

### `sys_permissions`

- 中文名称：权限点表
- 用途：定义页面、按钮、接口权限点
- 关键字段：
  - `permission_code`
  - `permission_name`
  - `module_code`
  - `permission_type`

### `sys_user_roles`

- 中文名称：用户角色关联表
- 用途：用户与角色多对多映射

### `sys_role_permissions`

- 中文名称：角色权限关联表
- 用途：角色默认权限模板

### `sys_user_permission_overrides`

- 中文名称：用户权限覆盖表
- 用途：额外授予 / 撤销用户权限
- 关键字段：
  - `effect_type`
  - `effective_from`
  - `effective_to`
  - `is_temporary`
  - `approval_status`

### `sys_data_scopes`

- 中文名称：数据范围表
- 用途：定义按区域 / 项目的数据权限范围

### `sys_role_data_scopes`

- 中文名称：角色数据范围关联表
- 用途：角色默认数据范围

### `sys_user_data_scope_overrides`

- 中文名称：用户数据范围覆盖表
- 用途：用户临时或特批的数据范围

### `sys_operation_logs`

- 中文名称：操作日志表
- 用途：记录系统关键写操作审计
- 关键字段：
  - `module_code`
  - `operation_type`
  - `operator_user_id`
  - `target_type`
  - `target_id`
  - `request_params_json`
  - `result_status`

### `sys_configs`

- 中文名称：系统配置表
- 用途：保存平台级配置项
- 关键字段：
  - `config_group`
  - `config_key`
  - `config_name`
  - `config_value_json`
  - `updated_by`

## 3. 业务对象层

### `biz_areas`

- 中文名称：区域表
- 用途：农场 / 温室 / 地块等业务区域
- 关键字段：
  - `area_code`
  - `area_name`
  - `area_type`
  - `crop_type`
  - `growth_stage`

### `iot_gateways`

- 中文名称：网关表
- 用途：存储网关或 ESP32 设备信息
- 关键字段：
  - `gateway_code`
  - `gateway_name`
  - `serial_no`
  - `online_status`
  - `wifi_rssi`
  - `cached_record_count`
  - `backfill_status`
  - `control_availability`

### `iot_sensors`

- 中文名称：传感器表
- 用途：环境与土壤传感器主数据
- 关键字段：
  - `sensor_code`
  - `sensor_type`
  - `modbus_address`
  - `current_value_decimal`
  - `sensor_status`
  - `calibration_status`

### `iot_actuators`

- 中文名称：执行器表
- 用途：水泵、电磁阀、风扇等执行器主数据
- 关键字段：
  - `actuator_code`
  - `actuator_type`
  - `control_channel`
  - `desired_state_text`
  - `reported_state_text`
  - `shadow_status`
  - `running_mode`

### `iot_device_shadow`

- 中文名称：设备影子状态表
- 用途：维护期望状态与实际状态
- 关键字段：
  - `desired_state_json`
  - `reported_state_json`
  - `shadow_status`
  - `desired_updated_at`
  - `reported_updated_at`
  - `last_command_id`
  - `last_command_result`
  - `drift_seconds`

### `iot_cameras`（规划）

- 中文名称：摄像头表
- 用途：存储 WiFi / IPC 摄像头基础信息
- 关键字段：
  - `camera_code`
  - `camera_name`
  - `camera_type`
  - `gateway_id`
  - `area_id`
  - `media_node_id`
  - `stream_protocol`
  - `online_status`
  - `snapshot_enabled`
  - `record_enabled`

### `iot_camera_streams`（规划）

- 中文名称：摄像头流配置表
- 用途：维护 RTSP/RTMP/HLS 等流配置
- 关键字段：
  - `camera_id`
  - `stream_role`
  - `source_protocol`
  - `source_url`
  - `target_protocol`
  - `target_url`

### `iot_media_nodes`（规划）

- 中文名称：媒体节点表
- 用途：管理转码、抓图、FTP 接收等媒体服务节点
- 关键字段：
  - `node_code`
  - `node_name`
  - `node_type`
  - `host_address`
  - `api_base_url`
  - `health_status`

## 4. 数据与运行层

  - `clock_synced`
  - `time_uncertainty_ms`

  - `delay_ms`

  - `finished_at`

- 关键字段：

  - `capture_job_id`
  - `source_type`
  - `captured_at`
  - `storage_provider`
  - `file_path`
  - `object_key`

### `iot_camera_recordings`（规划）

- 中文名称：摄像头录像索引表
- 用途：保存录像元数据与回看索引
- 关键字段：
  - `recording_no`
  - `camera_id`
  - `trigger_type`
  - `start_time`
  - `end_time`
  - `duration_seconds`
  - `file_path`
  - `status`

- 关键字段：

  - `rule_name`
  - `rule_type`
