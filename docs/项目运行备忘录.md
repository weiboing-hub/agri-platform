```bash

systemctl enable mosquitto
journalctl -u mosquitto -n 200 --no-pager
```

说明：

- 当前已关闭匿名访问
- 使用账号密码认证
- `/etc/mosquitto/passwd` 里是哈希，不能直接看回明文
- 明文连接信息已单独保存在 `/root/mqtt-broker-info.txt`

如果后面忘记密码：

1. 先执行 `cat /root/mqtt-broker-info.txt`
2. 如果记录丢失，再用下面命令重置：

```bash
mosquitto_passwd /etc/mosquitto/passwd esp32
systemctl restart mosquitto
```
